Avant que votre application puisse se connecter à une source de données, les pilotes JDBC du fournisseur de la source de données doivent être installés. Vous avez le choix entre deux méthodes d'installation des pilotes JDBC :

Modules

Pour installer un pilote JDBC en tant que module, vous devez créer une structure de chemin de fichier sous le répertoire ${build.installDir}/modules, copier le JAR du pilote JDBC dans le sous-répertoire main/ et créer un fichier module.xml.

Une fois que le pilote JDBC est disponible en tant que module, vous pouvez utiliser cette section pour ajouter, modifier et supprimer des configurations de pilotes.

Vous pouvez déployer des pilotes JDBC comme n'importe quel autre déploiement. Cela signifie que vous pouvez les déployer sur plusieurs serveurs dans un groupe de serveurs, si vous utilisez un domaine géré. Tout pilote compatible JDBC 4 sera automatiquement reconnu et installé dans le système par son nom et sa version.

En mode domaine, les pilotes déployés en tant qu'applications n'apparaîtront dans cette section que s'il y a des serveurs en cours d'exécution qui correspondent au profil sélectionné.
