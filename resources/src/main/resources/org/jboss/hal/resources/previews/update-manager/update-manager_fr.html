JBoss EAP fait l'objet de mises à jour périodiques qui contiennent des corrections de bogues et de sécurité. Il est recommandé de maintenir votre installation JBoss à jour.

Le gestionnaire de mise à jour vous indique si votre installation JBoss EAP existante nécessite des mises à jour. Il répertorie également toutes les mises à jour effectuées sur l'installation.

À partir de JBoss EAP 8.0, les fichiers de canaux sont utilisés pour définir les versions des composants du serveur JBoss EAP. Le gestionnaire de mise à jour fournit une liste des canaux associés à votre installation.

Vous pouvez également revenir à un état antérieur de l'installation.
