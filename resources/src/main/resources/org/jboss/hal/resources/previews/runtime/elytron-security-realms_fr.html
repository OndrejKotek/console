Fournit un ensemble d'opérations d'exécution pour les ressources suivantes du sous-système elytron :

  • Caching Realm: supprime toutes les entrées du cache.
  • Custom Modifiable Realm: ajouter, lire et supprimer des identités. Définit également un mot de passe pour une identité.
  • Filesystem Realm: ajouter, lire et supprimer des identités. Définit également un mot de passe pour une identité.
  • LDAP Realm: ajouter, lire et supprimer des identités. Définit également un mot de passe pour une identité.
  • Properties Realm: recharger les fichiers de propriétés à partir du système de fichiers.
