JBoss EAP has periodic releases which include:

  • Major release - new and enhanced features, bug fixes, security fixes, and all major architectural and configuration changes
  • Minor release - new and enhanced features, bug fixes and security fixes
  • Micro release (cumulative patches) - bug fixes and security fixes

You can update your existing installation of JBoss EAP in two ways:

  • Online updates - This option enables you to connect and get updates directly from an online Maven repository. You must have access to the internet to use the online option.
  • Offline using archive - This option enables you to perform the installation from a local or file-system Maven repository.
  • Custom patches - This options enables you to upload one or several custom patches.
