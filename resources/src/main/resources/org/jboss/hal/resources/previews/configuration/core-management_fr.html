Core-management configure les fonctions suivantes :

  • Changements de configuration : active ou désactive l'enregistrement en mémoire de tout changement de configuration de ce profil. Les changements de configuration pour la section du profil peuvent être visualisés dans la section d'exécution du serveur associé à ce profil.
  • Écoute de l'état du processus : ajoute une classe personnalisée pour écouter tout changement de l'état du processus du serveur.
