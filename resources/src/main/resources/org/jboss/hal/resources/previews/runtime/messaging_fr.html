Apache ActiveMQ Artemis

Apache ActiveMQ Artemis est un projet open source de système de messagerie asynchrone. Il est très performant, intégrable, en cluster et prend en charge plusieurs protocoles. ${build.shortName} utilise Apache ActiveMQ Artemis comme courtier JMS et est configuré à l'aide du sous-système messaging-activemq. Il remplace entièrement le courtier HornetQ mais conserve la compatibilité des protocoles avec les versions précédentes.

Le noyau d'ActiveMQ Artemis est agnostique au JMS et fournit une API non JMS, appelée API de base. ActiveMQ Artemis fournit également une API client JMS qui utilise une couche de façade pour implémenter la sémantique JMS au-dessus de l'API de base. Essentiellement, les interactions JMS sont traduites en opérations de l'API de base du côté client à l'aide de l'API client JMS. À partir de là, toutes les opérations sont envoyées à l'aide de l'API client de base et du format de fil Apache ActiveMQ Artemis. Le serveur lui-même n'utilise que l'API de base. Pour plus de détails sur l'API de base et ses concepts, reportez-vous à la Documentation ActiveMQ Artemis.
