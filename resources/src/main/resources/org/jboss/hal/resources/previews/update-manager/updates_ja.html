JBoss EAP には、バグとセキュリティーの修正を含む定期的なリリースがあります。JBoss EAP の既存のインストールは、 以下の方法で更新できます:

  • オンライン更新 - この方法を使用すると、オンライン Maven リポジトリーに接続して、そこから直接更新を取得できます。 オンライン更新を使用するには、インターネットにクセスできる必要があります。
  • アーカイブ使用したオフライン更新 - この方法を使用すると、ローカルまたはファイルシステムの Maven リポジトリーから インストールの更新を実行できます。
  • カスタムパッチ - この方法を使用すると、特別にカスタマイズしたパッチをインストールできます。

インストールを前の状態に戻すこともできます。
