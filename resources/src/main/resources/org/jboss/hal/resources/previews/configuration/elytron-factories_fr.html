Fabriques

Une fabrique d'authentification est une politique d'authentification utilisée pour des mécanismes d'authentification spécifiques. Les fabriques d'authentification sont spécifiquement basées sur le mécanisme d'authentification, par exemple http-authentication-factory et sasl-authentication-factory et kerberos-security-factory.

Transformateurs principaux

Un transformateur principal peut être utilisé à plusieurs endroits dans le sous-système elytron. Un transformateur principal peut transformer ou mettre en correspondance un nom avec un autre nom.
