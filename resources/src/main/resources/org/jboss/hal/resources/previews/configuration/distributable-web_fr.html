Le sous-système distribuable-web gère un ensemble de profils de gestion de session qui encapsulent la configuration d'un gestionnaire de session distribuable. L'un de ces profils sera désigné comme profil par défaut (via l'attribut "default-session-management") et définit donc le comportement par défaut d'une application web distribuable.

La gestion de session par défaut stocke les données de la session web dans un cache Infinispan.
