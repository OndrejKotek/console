Gérer les déploiements d'un groupe de serveurs

Vous pouvez utiliser la fonction drag and drop (glisser-deposer) pour ajouter de nouveaux déploiements au groupe de serveurs sélectionné. Il suffit de faire glisser un ou plusieurs fichiers sur la colonne de déploiement. Les déploiements ajoutés par glisser-déposer seront désactivés par défaut.

Sélectionnez un ou plusieurs éléments dans le référentiel de contenu et déployez-le dans le groupe de serveurs sélectionné.

Ajouter des déploiements non gérés. Un déploiement non géré pointe vers un dossier du système de fichiers local du serveur. Par rapport aux déploiements gérés, les déploiements non gérés ne seront pas copiés (c'est-à-dire téléchargés) dans le référentiel de déploiement du serveur avant d'être déployés. Le contenu du déploiement restera à son emplacement d'origine et sera déployé directement à partir de celui-ci.
