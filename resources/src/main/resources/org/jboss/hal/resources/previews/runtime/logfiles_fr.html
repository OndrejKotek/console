Visualiser les journaux du serveur et des applications afin d'aider à diagnostiquer les erreurs, les problèmes de performance et d'autres questions. Pour qu'un journal puisse être consulté, il doit se trouver dans le répertoire jboss.server.log.dir du serveur. La console respecte également les attributions de rôles RBAC des utilisateurs, de sorte qu'un utilisateur ne peut consulter que les journaux auxquels il est autorisé à accéder.

Par défaut, la console affiche les 2000 dernières lignes d'un fichier journal. Si vous souhaitez consulter le fichier journal dans son intégralité, choisissez "Télécharger". Pour visualiser ou suivre le fichier journal dans une fenêtre de navigation séparée, choisissez "Ouvrir dans une fenêtre externe".
