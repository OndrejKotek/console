L'architecture CORBA (Common Object Request Broker Architecture) est une norme qui permet aux applications et aux services de fonctionner ensemble, même lorsqu'ils sont écrits dans des langages multiples, autrement incompatibles, ou hébergés sur des plates-formes distinctes. Les requêtes CORBA sont transmises par un composant côté serveur appelé "Object Request Broker" (ORB). ${build.shortName} fournit une instance d'ORB au moyen du composant Open JDK ORB.
