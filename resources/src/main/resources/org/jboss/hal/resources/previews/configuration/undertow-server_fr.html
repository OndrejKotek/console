Un serveur représente une instance de Undertow et se compose de plusieurs éléments :

  1. Hôte
  2. HTTP Listener
  3. HTTPS Listener
  4. AJP Listener

L'élément hôte fournit une configuration d'hôte virtuel tandis que les trois listeners fournissent des connexions de ce type à l'instance Undertow.

Remarque
Plusieurs serveurs peuvent être configurés, ce qui permet d'isoler complètement les déploiements et les serveurs. Cela peut s'avérer utile dans certains scénarios tels que les environnements multi-locataires.
