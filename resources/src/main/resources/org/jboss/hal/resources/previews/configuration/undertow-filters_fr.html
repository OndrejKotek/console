Un filtre permet de modifier certains aspects d'une requête et peut utiliser des prédicats pour contrôler l'exécution du filtre. Parmi les cas d'utilisation courants des filtres, on peut citer la définition d'en-têtes ou la compression GZIP.

Les types de filtres suivants peuvent être définis :

  1. Filtre personnalisé
  2. Page d'erreur
  3. Filtre d'expression
  4. GZip
  5. Mod Cluster
  6. Limite de demande
  7. En-tête de réponse
  8. Réécriture

Remarque
Un filtre est fonctionnellement équivalent à une valve globale utilisée dans les versions précédentes de ${build.shortName}.
