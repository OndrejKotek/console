Le serveur web (Undertow) contient de nombreuses métriques d'exécution, réparties dans les sections suivantes :

  • Domaine de sécurité des applications: affiche la configuration du domaine de sécurité de l'application et ses déploiements associés.
  • Serveur: affiche les métriques d'exécution des connecteurs enregistrés comme ajp-listener, http-listener et https-connectors d'un serveur spécifique.
  • Modcluster: affiche les métriques d'exécution pour les équilibreurs, les nœuds et les contextes.
  • Déploiement: affiche les métriques d'exécution pour les servlets et les websockets contenus dans les déploiements.
