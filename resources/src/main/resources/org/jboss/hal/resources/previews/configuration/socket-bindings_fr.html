Une liaison de socket est une configuration nommée pour un socket. Les configurations domain.xml et standalone.xml comprennent toutes deux une section dans laquelle des configurations de socket nommées peuvent être déclarées. D'autres sections de la configuration peuvent alors faire référence à ces sockets par leur nom logique, plutôt que d'avoir à inclure tous les détails de la configuration de la socket (qui peut varier sur différentes machines).
