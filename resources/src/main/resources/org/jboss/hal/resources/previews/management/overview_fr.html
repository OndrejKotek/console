Configurer les paramètres liés à l'interface de gestion, revoir les derniers changements de configuration et gérer les extensions de la console de gestion.

Interface de gestion

Visualiser et modifier les paramètres des interfaces de gestion http et natives comme la liste des origines autorisées, le domaine de sécurité, le contexte SSL.

Extensions

Visualiser et gérer les extensions de la console de gestion. Les extensions permettent d'ajouter de nouvelles fonctionnalités à la console de gestion. Elles sont écrites en JavaScript et se divisent en deux catégories :

  • Extensions groupées : partie de l'installation de ${build.fullName} fournie par les modules
  • Extensions autonomes : extensions tierces hébergées par un serveur public
