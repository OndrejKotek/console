Contient la configuration pour découvrir les groupes, les connecteurs, les fabriques de connexion, les files d'attente et les sujets vers un serveur ActiveMQ Artemis distant :

Connecteurs

Un connecteur définit comment se connecter à un serveur ActiveMQ Artemis distant.

Pour plus d'informations sur les connecteurs, voir la Documentation sur les connecteurs.

Groupes discovery

Un groupe discovery définit la manière dont les informations sur les connecteurs sont reçues à partir d'un point d'extrémité de diffusion, par exemple une adresse multicast UDP ou un canal JGroup.

Un groupe discovery maintient une liste de connecteurs, un pour chaque diffusion par un serveur différent. Lorsqu'il reçoit des diffusions sur le point d'extrémité de diffusion d'un serveur particulier, il met à jour son entrée dans la liste pour ce serveur. S'il n'a pas reçu de diffusion d'un serveur particulier pendant un certain temps, il supprime l'entrée de ce serveur de sa liste.

Fabriques de connexion

Par défaut, le sous-système de messagerie ${build.shortName} fournit les fabriques de connexion InVmConnectionFactory et RemoteConnectionFactory, ainsi que la fabrique de connexion mutualisée activemq-ra pooled connection factory.

Les fabriques de connexions mutualisées vous permettent de configurer les connecteurs entrants et sortants de l'adaptateur de ressources ActiveMQ Artemis distant. Pour plus d'informations sur la configuration d'une fabrique de connexions groupées pour se connecter à un serveur ActiveMQ Artemis distant, voir Utilisation de l'adaptateur de ressources intégré pour les connexions distantes.

File d'attente externe / Sujet

Les files d'attente et les sujets qui existent dans le serveur ActiveMQ Artemis distant.
