Les workers sont des instances de travailleurs XNIO. Une instance de worker XNIO est une couche d'abstraction pour les API Java NIO, qui fournissent des fonctionnalités telles que la gestion des IO et des threads de worker ainsi que la prise en charge de SSL. Par défaut, ${build.shortName} fournit un seul worker appelé default, mais il est possible d'en définir d'autres.
