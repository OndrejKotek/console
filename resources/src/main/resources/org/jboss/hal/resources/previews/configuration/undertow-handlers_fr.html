Undertow permet de configurer deux types de gestionnaires ou handlers :

  1. Gestionnaire de fichiers
  2. Gestionnaires de proxy inversé (reverse-proxy)

Les gestionnaires de fichiers servent les fichiers statiques. Chaque gestionnaire de fichiers doit être attaché à un emplacement dans un hôte virtuel. Les gestionnaires de reverse-proxy permettent à ${build.shortName} d'utiliser le serveur comme un reverse-proxy très performant. Il peut gérer les backends AJP, HTTP et HTTP.2 et prend en charge mod_cluster.
