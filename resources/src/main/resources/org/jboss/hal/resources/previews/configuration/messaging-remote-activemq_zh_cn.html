包含发现组、连接器、连接工厂、队列和主题到远程 ActiveMQ Artemis 服务器的配置：

连接器

连接器定义如何连接到一个远程 ActiveMQ Artemis 服务器。

有关连接器的更多信息，请参阅 连接器文档.

发现组（discovery group）

广播组定义了如何从一个广播端点（例如，一个 UDP 多播地址或 JGroup 信道）接收连接器信息。

发现组维护一个连接器列表，每个连接器用于由一个不同的服务器发送的每个广播。由于它从特定服务器接收广播端点上的广播，它会更新该服务器列表列表中的条目。如果它在一定时间内尚未从特定服务器收到广播，会从列表中删除该服务器的条目。

连接工厂

默认情况下，${build.shortName} 消息子系统提供了 InVmConnectionFactory 和 RemoteConnectionFactory 连接工厂，以及 activemq-ra 池的连接工厂。

池化连接工厂允许您配置远程 ActiveMQ Artemis 资源适配器的入站和出站连接器。如需了解更多与配置 pooled-connection-factory 连接到远程 ActiveMQ Artemis 服务器的信息，请参阅使用集成资源适配器进行远程连接。
