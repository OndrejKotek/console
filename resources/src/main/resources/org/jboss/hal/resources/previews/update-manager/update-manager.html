JBoss EAP has periodic releases that contain bug and security fixes. It is recommended to keep your JBoss installation up-to-date.

Update manager provides you information on whether your existing JBoss EAP installation requires updates. It also lists all the updates done on the server.

From JBoss EAP 8.0 onward, channel files are used to define the versions of the JBoss EAP server artifacts. The Update manager provides a list of channels associated with your installation.
